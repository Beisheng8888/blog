    内部类和Lambda表达式

    1.内部类

    1.1 内部类的基本使用（理解）

    • 内部类概念

      • 在一个类中定义一个类。举例：在一个类A的内部定义一个类B，类B就被称为内部类

    • 内部类定义格式

      • 格式&举例：

        /*
	格式：
    class 外部类名{
    	修饰符 class 内部类名{
    	
    	}
    }
*/

class Outer {
    public class Inner {
        
    }
}

    • 内部类的访问特点

      • 内部类可以直接访问外部类的成员，包括私有
      • 外部类要访问内部类的成员，必须创建对象

    • 示例代码：

      /*
    内部类访问特点：
        内部类可以直接访问外部类的成员，包括私有
        外部类要访问内部类的成员，必须创建对象
 */
public class Outer {
    private int num = 10;
    public class Inner {
        public void show() {
            System.out.println(num);
        }
    }
    public void method() {
        Inner i = new Inner();
        i.show();
    }
}
